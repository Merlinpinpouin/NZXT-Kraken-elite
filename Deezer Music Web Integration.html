Bien sûr, voici un exemple simple de code JavaScript pour récupérer les données de chanson depuis l'API Deezer et mettre à jour les informations affichées dans votre intégration web. Dans cet exemple, j'utilise l'API JSONP de Deezer pour effectuer une requête. Assurez-vous de lire la documentation Deezer API pour comprendre les détails de l'authentification et de l'utilisation des endpoints API.

html
<!DOCTYPE html>
<html lang="en">
<head>
    <!-- Meta tags, title, and CSS styles here -->
</head>
<body>
    <div class="music-info">
        <img class="album-cover" src="" alt="Album Cover">
        <div class="song-title"></div>
        <div class="artist-name"></div>
    </div>

    <script>
        const appId = '628564';
        const secretKey = '3d71987056ebd1d4b2d44e413c951c22';

        function fetchDeezerData() {
            const deezerApiUrl = `https://api.deezer.com/user/${appId}/tracks?secret=${secretKey}&limit=1`;

            // Make a JSONP request to Deezer API
            const script = document.createElement('script');
            script.src = deezerApiUrl;
            document.body.appendChild(script);
        }

        // Function called by JSONP response from Deezer API
        function callbackDeezerData(data) {
            const musicInfo = data[0];

            const albumCover = document.querySelector('.album-cover');
            const songTitle = document.querySelector('.song-title');
            const artistName = document.querySelector('.artist-name');

            albumCover.src = musicInfo.album.cover_medium;
            songTitle.textContent = musicInfo.title;
            artistName.textContent = musicInfo.artist.name;
        }

        fetchDeezerData();
    </script>
</body>
</html>

Points importants à noter :

  • J'utilise JSONP pour effectuer une requête à l'API Deezer. Cela nécessite d'ajouter un <script> à la page avec l'URL de l'API. Lorsque la réponse de l'API est reçue, la fonction callbackDeezerData est appelée.
  • Assurez-vous d'ajuster l'URL de l'API Deezer en fonction des endpoints et des paramètres requis.
  • Les données de la chanson sont extraites de la réponse JSON de l'API et utilisées pour mettre à jour les éléments HTML correspondants.

N'oubliez pas de consulter la documentation Deezer API pour obtenir des informations précises sur la structure des données et les méthodes d'authentification.
